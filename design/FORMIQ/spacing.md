# Spacing

## Principles

- Use air as a design material.
- Interfaces should feel precise, not crowded.
- Panels float over the map with clear breathing room.
- Avoid nested cards and heavy boxed layouts.

## Recommended Scale

Use an 8px-based spacing rhythm:

- 4px: tight icon/text alignment
- 8px: compact internal gaps
- 12px: dense control groups
- 16px: default component padding
- 20px: glass panel padding
- 24px: major panel padding
- 32px: section spacing
- 48px: large composition spacing
- 64px: screen-level breathing room

## Layout Rules

- Keep the map visible around and between panels.
- Prefer floating panels to full-height sidebars when the workflow allows it.
- Use diagonal composition only when it supports spatial hierarchy or map focus.
- Do not fill all edges with UI.
- Keep tool surfaces stable: hover states must not resize controls.
