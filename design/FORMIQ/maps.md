# Maps

## GIS-First Rule

The map is the main element of FORMIQ. All GIS workspace UI is built over the map.

## Map Composition

- The map should occupy the primary viewport.
- Panels should float over the map using glass surfaces.
- Keep enough visible map area for spatial orientation.
- Avoid heavy sidebars unless a workflow requires persistent structure.
- Use diagonal composition to guide attention across the map when useful.

## Tooling

Map controls should be:

- Compact
- Icon-led
- Clearly grouped
- Spatially close to the map operation they affect

## Data Layers

- Layer controls must not visually dominate the canvas.
- Use clear active states and small metadata.
- Complex layer settings should open in a focused panel instead of expanding endlessly.

## Project Home Exception

Project-home screens are not GIS workspaces. They manage projects only and must not show GIS tools, map controls, source controls, analysis tools, layers, or terrain tools.
