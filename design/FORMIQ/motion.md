# Motion

## Motion Principles

- Motion must support clarity.
- Use short, calm transitions.
- Avoid playful or decorative animation.
- GIS interactions must feel precise and responsive.

## Defaults

```css
transition: 200ms ease-out;
```

Hover:

```css
transform: translateY(-2px);
```

## Allowed Motion

- Panel reveal and dismissal
- Tool hover and active states
- Map tool selection feedback
- Loading progress for import, export, analysis, and 3D generation
- Subtle opacity and blur transitions

## Avoid

- Bouncy easing
- Oversized hover movement
- Decorative background animation
- Motion that competes with map navigation
