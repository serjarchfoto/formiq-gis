# Presentation

## Purpose

Presentation mode turns GIS work into a clear architectural story.

## Visual Direction

- Large typography
- Clean hierarchy
- Minimal colors
- Map and spatial model as the hero
- Glass controls that do not hide the content

## Rules

- Keep the map or generated view visible as the primary visual.
- Controls should collapse when presenting.
- Use labels and legends sparingly.
- Use large background words only when they support the presentation and remain low-contrast.

## Export Surfaces

Export previews must look consistent with the workspace:

- Same typography
- Same primary color
- Same legend style
- Same restrained panel language
