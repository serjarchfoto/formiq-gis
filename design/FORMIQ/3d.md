# 3D

## Purpose

3D views support architectural and urban analysis while preserving the GIS-first workspace.

## Composition

- 3D canvas is a primary visual surface, like the map.
- Controls float over the scene as glass panels.
- Avoid framed preview cards for the main 3D scene.
- Keep navigation, camera, layer, and export tools compact.

## Visual Style

- Clean architectural rendering
- Minimal color
- Clear silhouettes
- Subtle atmospheric depth only when it improves readability
- No decorative effects that reduce spatial clarity

## Controls

- Camera controls: compact icon group
- Layer and massing controls: glass panel
- Export controls: focused panel or menu
- Legends: small, readable, and non-obstructive
