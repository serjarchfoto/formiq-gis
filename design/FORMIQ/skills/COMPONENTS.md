# Components

## Shared Style

Use one visual language across the product:

```css
background: rgba(255, 255, 255, 0.62);
backdrop-filter: blur(24px);
border: 1px solid rgba(255, 255, 255, 0.35);
box-shadow: none;
transition: transform 200ms ease-out, background 200ms ease-out, border-color 200ms ease-out;
```

## Buttons

- Border radius: `14px`
- Primary button: `#229ED9` background, white text
- Secondary button: glass background, primary text
- Destructive button: use `#EF4444` only when explicit

## Panels

- Border radius: `20px`
- Use glass surfaces.
- Do not use heavy card stacks.

## Controls

- Use icons when the action is tool-like or spatial.
- Use text when clarity is more important than compactness.
- Keep hover and active states restrained.
