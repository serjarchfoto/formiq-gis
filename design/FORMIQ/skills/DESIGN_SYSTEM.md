# FORMIQ Design System

This folder contains the operational design guidance for FORMIQ.

## Source Priority

1. `design/FORMIQ`
2. `design/open-design`
3. Existing project code

## Core Principle

The map is the main interface element. UI sits over the map and supports it.

## Baseline Rules

- Use Inter Variable for interface text.
- Use General Sans Bold for the logo.
- Prefer glass panels over heavy cards.
- Use thin borders and minimal shadows.
- Keep typography large and confident.
- Keep the interface calm, precise, and GIS-first.
