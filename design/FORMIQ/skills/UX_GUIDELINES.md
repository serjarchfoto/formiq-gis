# UX Guidelines

## Product Model

FORMIQ is a GIS and architectural planning tool. The workflow should feel spatial, direct, and efficient.

## UX Rules

- Keep the map visible and primary.
- Use overlays for tools, analysis, import, export, and 3D.
- Avoid crowded layouts and unnecessary steps.
- Prefer single-action flows when the task can be completed that way.
- Keep labels short, direct, and professional.

## Interaction Principles

- Selection should be obvious.
- Feedback should be immediate.
- Motion should clarify state, not decorate it.
- Do not expose internal system complexity to the main user flow.
