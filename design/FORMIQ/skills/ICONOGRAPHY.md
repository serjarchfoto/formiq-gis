# Iconography

## Rules

- Use clear, familiar icons for tool actions.
- Prefer icons inside buttons when the action is spatial or tool-like.
- Keep icon treatment consistent across the product.
- Use icons to support scanning, not decoration.

## Guidance

- Pair icons with text when the action might be ambiguous.
- Keep icon size balanced with 16px body UI.
- Avoid decorative icon sets that feel generic or noisy.
