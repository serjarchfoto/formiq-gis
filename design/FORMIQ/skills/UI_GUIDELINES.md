# UI Guidelines

## Visual Style

- Glass panels.
- Thin borders.
- Minimal shadows.
- Airy composition.
- Large typography.
- Map-first layout.

## Panel Rules

- Panels should feel like overlays, not separate pages.
- Avoid heavy stacked cards.
- Keep controls compact and readable.
- Use restrained emphasis for active states.

## Interaction Defaults

- Hover: `transform: translateY(-2px);`
- Transition: `200ms ease-out`
- Panel blur: `24px`
